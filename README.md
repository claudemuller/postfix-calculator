# Postfix Calculator

A postfix/reverse polish notation calculator implemented in JavaScript

## Installation

  `npm install @claudemuller/postfix-calculator`

## Usage

   ```
   const calculate = require('@claudemuller/postfix-calculator');

   var result = calculate(5, 6, 7, '*', '+', 1, '-');
   ```
  Output should be `46`

## Tests
